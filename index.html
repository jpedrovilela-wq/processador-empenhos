Processador de Empenhos

Envie a planilha, processe os empenhos e baixe uma cópia pronta. O arquivo é tratado somente neste navegador — não é enviado para nenhum servidor.

Selecione uma planilha para começar.

O que o processamento faz?
  • Relaciona os dados exclusivamente pelo número do contrato.
  • Prioriza empenhos de 2025 e depois os de 2026.
  • Cria a aba LOG para casos sem correspondência ou com saldo insuficiente.
  • No arquivo final, a coluna M contém o Valor distribuído e a coluna O preserva o Valor total do Contrato original.
  • Após o download, você pode selecionar outro arquivo — inclusive o mesmo arquivo original — para iniciar um novo processamento.
